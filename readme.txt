Instructions to clone, push, and edit this repository:

[1] First you need to get the repository cloned to your working directory.

$ git clone https://github.com/edi-mcgill/edi.git

[2] Make your wonderful modifications ;)

[3] In your terminal you can test your development by the following. eg.

$ firefox index.html
$ google-chrome index.html

[4] Add and commit your changes to the repository. Use a clear message
when committing.

$ git add mynewIndex.html
$ git commit -m "my new awesome home page"

[5] Push your code.

$ git push -u origin master


This is a bare bones guide to getting started maintaining and upkeeping
the website. It is good to check the website from time to time and check
that the facebook link is working properly. I've found that the code for 
this part in particular changes more often.
